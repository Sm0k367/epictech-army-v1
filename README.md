## 🚀 Overview

Epic Tech AI is a barrier-smashing, public, zero-cost creative playground:
- 🎵 Make music, write lyrics/poetry, generate art & logos — all free, no keys required
- 📖 Instantly code, remix, or challenge the Agent Army
- 🛠 Build, deploy, export, and play — browser-first & always evolving

Live Demo: [https://epictech-army-v1.vercel.app](https://epictech-army-v1.vercel.app)

---

## 🔥 Features

- **Creative Labs**:
- Music, Text, Logo — browser-native, unlimited use
- **Showcase Arena**:
- Agent Army divisions, Creative Gallery
- **Interactive Challenge**:
- Throw any creative prompt, watch AI “army” reply live
- **Instant Feedback & Social Share**:
- Share to X/Twitter, give feedback instantly
- **BYOK/-Bring Your Own Key-**:
- (Planned) — plug in your own AI keys for advanced features

## 🛠️ Tech Stack

- Next.js (App Router, TypeScript)
- TailwindCSS
- Tone.js, React Icons, Browser APIs (no backend required)
- Vercel Deployment
