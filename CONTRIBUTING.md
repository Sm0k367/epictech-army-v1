# Contributing to Epic Tech AI Agent Army

🚀 **Welcome, creative hackers, artists, and inventors!**

We want your wildest PRs—feature labs, bug fixes, docs, or experimental modules.
Whether you’re remixing a component, adding a free AI API, or just fixing a typo, you’re part of the Agent Army.

## How to Contribute

1. **Fork the repo** and create a feature branch.
2. **Commit changes** with clear, direct messages.
3. **Make a pull request**—describe your experiment, improvement, or fix.
4. **Be bold:** We love unconventional features!
5. **Talk it out:** Issues and ideas welcome for collaboration and requests.

## Coding Style

- Use TypeScript for all React/Next components.
- Prefer composition and modularity—think “labs” and “divisions.”
- Keep core creative tools browser-native and zero-cost by default.
- Docs and demo code are as valued as features!

## Code of Conduct

Be helpful, direct, and respect the purpose:
**Build the impossible, help others do the same, never gatekeep.**
